This is freg help file. Game commands:
'Q'       - quit game
arrows    - move in four directions
End       - move down
'>', '<'  - turn right and left
space     - jump
Page Up   - look up
Page Down - loo down
Home      - inventory
Backspace - hit
Enter     - use
'?'       - examine
'~'       - inscribe
'!'       - enter creative mode
':'       - enter command mode
'.'       - repeat last command
'L'       - redraw screen
'{', '}'  - shift focus down and up
'-', '+'  - decrease and increase active zone
'_', '#'  - decrease and increase world size
Set inventory actions:
'B' - build
'C' - craft
'E' - eat
'F' - takeoff (unwield)
'H' - show this help
'I' - inscribe
'U' - use
'O' - obtain from outer inventory (e.g. chest)
'T' - throw away
'W' - wield
'a'-'z' - make inventory action with this inventory slot
