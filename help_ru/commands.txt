 Команды freg. [!] - доступно только в режиме Творца.
 help PAGE
 показать страницу справки PAGE.

 give KIND SUB NUMBER (== get KIND SUB NUMBER) [!]
 * KIND — код запрошенного блока (по умолчанию 0 - блок).
 * SUB — код запрошенного вещества (по умолчанию 0 - камень).
 * NUMBER — количество запрошенных блоков (по умолчанию - 1).
 Посмотреть коды типов и веществ: help kinds, help subs

 move DIRECTION
 * DIRECTION — направление движения. Доступные направления:
 * 0,1,2,3,4,3: вверх (по умолчанию), вниз, на север, на юг,
 на восток, на запад.

 what X Y Z — исследовать блок.
 * X, Y, Z — координаты исследуемого блока.

 kindtostring CODE (== k2str)
 Показать, какому типу соответствует код типа CODE.

 subtostring CODE (== s2str)
 Показать, какому веществу соответствует код вещества CODE.

 stringtokind STRING (== str2k)
 Показать, какому коду типа соответствует строка STRING.

 stringtosub STRING (== str2s)
 Показать, какому коду вещества соответствует строка STRING.

 time [!]
 Показать время.
