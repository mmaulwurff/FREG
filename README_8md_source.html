1 About {#mainpage}
2 =====
3 
4 FREG is a free-roaming elementary game.
5 Inspired by Minecraft and Angband.
6 This is a version of FREG with text graphics.
7 
8 Development stage: implementing basics.
9 
10 Language: C++11, Qt (version >= 5.3)
11 
12 Non-standard libraries: libncurses5
13 
14 Development is run on GNU/Linux, but on MS Windows should work too with
15 PDCurses ( http://pdcurses.sourceforge.net/ ).
16 
17 Now only singleplayer is developed, multiplayer is planned.
18 
19 Compiling
20 ---------
21 
22  qmake freg-nox.pro;
23  make;
24 
25 Or open freg-nox with QtCreator.
26 
27 Map editing
28 -----------
29 
30 You can edit map.txt files in your text editor.
31 `map.txt` should be square, with CR line endings only.
32 There is Vim highlight file `fregMap.vim`. You can copy it in `~/.vim/syntax`
33 and set file type with command:
34 `:set filetype=fregMap`
35 
36 Also on large maps try option
37 `:set nowrap`
38 
39 Extra fun: block copy (Ctrl-v, then select, then p) and block paste (1vp).
