Detailed Description

This class is used for creating and deleting blocks, also for loading them from file.

Memory management, if any, should be implemented in this class. At the current moment no special memory management is used.

Normal blocks: blocks that are not special, e.g. usual stone, air, soil are actually one block (for each substance). One can receive a pointer to such block with Block * NormalBlock(int sub). Normal blocks are not needed to be deleted. Use Block * NewBlock(int kind, int sub) to receive a pointer to block that will be changed (damaged, inscribed, etc).
